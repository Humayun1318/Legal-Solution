A media query is a CSS technique that allows the presentation of a document to be tailored to a specific range of output devices without changing the content of the document itself. Media queries consist of a media type and one or more expressions that check for the conditions of particular media features. They can be used to apply different styles to different screen sizes, resolutions, and other characteristics of the output device, making a website more responsive and adaptable to different devices. For example, a media query can be used to change the font size of text on a small screen, or to hide certain elements on a mobile device that are displayed on a desktop.
